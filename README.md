# Hindley-Milner Type Inference in Haskell

This project implements a simple **Hindley-Milner type inference** algorithm in Haskell.

## Features

- Lambda expressions (`Var`, `Lam`, `App`)
- Type variables and function types
- Polymorphic type schemes (`Forall`)
- Type unification with occurs check
- Type inference following the Hindley-Milner algorithm
